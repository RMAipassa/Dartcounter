Drop pre-generated callout mp3 files here to improve voice quality.

Folder structure:
- /audio/callouts/en/

Supported file keys:
- score-0.mp3 ... score-180.mp3 (also supports 0.mp3 ... 180.mp3)
- bust.mp3
- game-shot.mp3
- game-shot-match.mp3
- game-shot-leg-1.mp3 ... game-shot-leg-20.mp3


Examples:
- /audio/callouts/en/score-180.mp3
- /audio/callouts/en/game-shot-leg-1.mp3

If a file is missing, Dartcounter falls back to browser speech synthesis.
